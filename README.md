## 🚀 What This Bot Can Do

| Feature | Description |
|:--------|:------------|
| 🎵 **High-Quality Music Streaming** | Play audio from YouTube, Spotify, and more with instant playback. |
| 🎬 **Video Playback in Voice Chats** | Stream music videos directly into your group calls. |
| 📋 **Smart Queue Management** | Add, skip, shuffle, and loop tracks with ease. |
| 🎮 **Built-in Fun & Games** | Challenge friends with Trivia, Guessing Game, and Slots. |
| 🌐 **Web Dashboard** | Monitor stats and manage the bot from a browser. |
| 🤖 **AI-Powered Features** | AI-generated music and lyrics for a modern twist. |
| 📊 **Live Bot Analytics** | Track user engagement and system performance in real time. |

## 📖 Complete Command Guide

| Command | Description | Example |
|:--------|:------------|:--------|
| `/start` | Start the bot and view welcome message | `/start` |
| `/help` | Display the help menu | `/help` |
| `/ping` | Check bot response time | `/ping` |
| `/stats` | View bot statistics | `/stats` |
| `/dashboard` | Get web dashboard link | `/dashboard` |
| `/play <song>` | Play music in voice chat | `/play Believe` |
| `/vplay <song>` | Play music video | `/vplay Shape of You` |
| `/pause` | Pause current playback | `/pause` |
| `/resume` | Resume paused playback | `/resume` |
| `/skip` | Skip to next song | `/skip` |
| `/stop` | Stop playback and clear queue | `/stop` |
| `/queue` | Show song queue | `/queue` |
| `/trivia` | Start a trivia game | `/trivia` |
| `/guess` | Number guessing game | `/guess` |
| `/slots` | Slot machine game | `/slots` |
| `/radio` | Play live radio stream | `/radio BBC` |

## 📚 Environment Variables

| Variable | Description |
|:--------|:------------|
| `API_ID` | Your Telegram API ID from my.telegram.org |
| `API_HASH` | Your Telegram API hash from my.telegram.org |
| `BOT_TOKEN` | Your bot token from @BotFather |
| `STRING_SESSION` | Pyrogram string session of your assistant account |
| `OWNER_ID` | Your Telegram user ID |
| `MONGO_URI` | MongoDB connection URI |
| `START_IMAGE_URL` | URL of your start message image |
| `STICKER_ID` | Telegram sticker file ID |
